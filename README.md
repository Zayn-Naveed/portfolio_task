# Dynamic Home Page with Hero Slider

A modern, responsive portfolio homepage featuring an animated hero slider.

## Live Demo
[View Live Site](https://creative-dieffenbachia-42a3d1.netlify.app/)

## GitHub Repository
[View Code](https://github.com/Zayn-Naveed/portfolio_task)

## Technologies Used
- HTML5 (Semantic)
- CSS3 (Flexbox, Animations, Responsive Design)
- JavaScript (Vanilla JS)
- Swiper.js (Hero Slider Library)

## Features
- Full-width animated hero slider
- Auto-rotates every 4.5 seconds
- Previous/Next arrow navigation
- Dot indicators
- Keyboard navigation (Left/Right arrow keys)
- Smooth fade transitions
- Hover effects on buttons
- Fully responsive (Desktop, Tablet, Mobile)
- Semantic HTML5 structure
- Alt text on all images for accessibility

## Project Structure

## Design Choices
- Dark theme for modern, professional look
- Cyan (#00d4ff) accent color for highlights
- Glassmorphism navbar effect
- Fade transition for smooth slide changes
- Picsum placeholder images used for demonstration
